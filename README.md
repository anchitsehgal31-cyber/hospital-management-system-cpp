 Hospital Management System (C++)

 Description
A console-based Hospital Management System developed using C++ and Object-Oriented Programming (OOPS) concepts.

 Features
- Add Doctor and Patient records
- Book Appointments
- Generate Billing
- Display Doctors, Patients, and Appointments

 OOPS Concepts Used
- Classes & Objects
- Inheritance
- Polymorphism
- Encapsulation
- Multi-file Project Structure

 How to Run
1. Compile all files:
   g++ main.cpp Person.cpp Doctor.cpp Patient.cpp Appointment.cpp Billing.cpp HospitalSystem.cpp -o hospital
2. Run:
   ./hospital   (Linux/Mac)
   .\hospital.exe (Windows)
